## API Endpoints

### POST `/api/captions/generate`

Generates 5 AI-powered captions

**Request:**

```json
{
  "contentInput": "string",
  "contextData": {
    "productLink": "string",
    "uploadedImage": "File",
    "selectedPreviousPost": "string"
  },
  "selectedContextItems": ["string"],
  "options": {
    "cta": "string",
    "hashtagCount": "number",
    "captionLength": "string",
    "emojiStyle": "string",
    "customCta": "string"
  }
}
```

**Response:**

```json
{
  "success": true,
  "captions": ["string", "string", ...],
  "platform": "string"
}
```

### GET `/api/captions/recent?limit=10`

Fetches user's recent captions

**Response:**

```json
{
  "success": true,
  "captions": [
    {
      "id": "string",
      "snippet": "string",
      "fullText": "string",
      "date": "string",
      "platform": "string",
      "style": "string",
      "metadata": {}
    }
  ]
}
```
